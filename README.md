# ITIL 4 Foundation Practice

A self-contained, offline-ready practice app for the **ITIL® 4 Foundation** certification exam. 120 original questions across all five syllabus topic areas, with three study modes, instant or deferred grading, and a detailed end-of-session report.

**[▶ Launch the app](https://ranrhoads84.github.io/itil4-foundation-practice/)**

---

## Features

### Three Study Modes

| Mode | Description |
|------|-------------|
| **Study** | Instant feedback + explanations after every answer. Pick your topics and length. |
| **Exam Simulation** | 40 questions drawn on the official Foundation weighting, 60-minute countdown, no feedback until the end. |
| **Topic Drill** | Hammer a single topic area with instant feedback. |

### Question Bank — 120 Questions

| Topic | Questions |
|-------|-----------|
| Key Concepts | 22 |
| Guiding Principles | 26 |
| Four Dimensions | 16 |
| SVS & Value Chain | 23 |
| Practices | 33 |

Every question includes a **difficulty tag** (recall / application) and a **full explanation**.  
Questions and answer-choice order are **reshuffled every run**.

### Exam Simulation Weighting

Matches the official ITIL 4 Foundation syllabus distribution:

| Area | Questions (of 40) |
|------|------------------|
| Key Concepts | 7 |
| Guiding Principles | 9 |
| Four Dimensions | 4 |
| SVS & Value Chain | 7 |
| Practices | 13 |

### Results Report

- Pass / fail verdict against the real **65% threshold** (26/40)
- **Per-topic breakdown** based on answered questions only — no phantom zeroes for untouched topics
- In Exam Sim, shows how many questions were drawn per area that run
- **Weakest area** callout
- Review mode: **All**, **Incorrect only**, or **⚑ Flagged only**

### Flag for Review

Mark any question with the **⚑ flag button** (or press **F**) to revisit it later.  
Flags persist across back/next navigation in Exam Sim mode.  
At results, filter the review list to flagged questions only.

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `1` `2` `3` `4` or `A` `B` `C` `D` | Select answer |
| `Enter` | Next question |
| `F` | Toggle flag |
| `←` | Previous question (Exam Sim) |

## Usage

**Online:** click the [launch link](https://ranrhoads84.github.io/itil4-foundation-practice/) above.

**Offline:** download [`index.html`](./index.html) and open it in any browser. No server, no dependencies, no internet required — everything is embedded in the single file.
